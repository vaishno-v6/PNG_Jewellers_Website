# PNG_Jewellers_Website
PNG_Jewellers_Website

1. Project Overview:
- PNG Jewellers ek responsive jewellery website hai.
- Users ko jewellery collections, offers aur contact information easily dekhne ko milti hai.

2. Technologies Used:

 HTML5 – Website ka structure create karne ke liye.
 CSS3 – Styling aur design ke liye.
 Bootstrap 5 – Responsive layout aur components ke liye (navbar, carousel, grid).
 JavaScript (Optional) – Interactive features ke liye (agar use kiya ho).

3. Features:
 - Responsive design for mobile, tablet, aur desktop.
 - Product gallery with hover effects.
 - Bootstrap carousel for featured jewellery.
 - Contact form with basic validations.

4. How to Download:

 - GitHub ya ZIP file se project folder download kare.
 - ZIP file extract kare apne computer me.

5. How to Open / Run:

 - Project folder me PNGByVaishnavi.html file locate kare.
 - Double-click kare ya right-click → “Open with” → browser select kare (Chrome/Firefox/Edge).
 - Website browser me open ho jayegi aur navigate kar sakte ho.
